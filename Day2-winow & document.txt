WINDOW OBJECT:
           It is the topmost object of the DOM hierarchy
           It represent the browser
           window or frame that display contents webpage
           It is very first object that is loaded in the browser
           It is the object of the browser
           Global object function and variables of javascript are members of widow object
           EX:
             window document title will return that of document
